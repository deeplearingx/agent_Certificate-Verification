## 模块功能说明

### 1. Config - 配置管理
```python
from core import Config

print(Config.DB_DIR)          # 获取数据库路径
print(Config.MODEL)           # 获取模型名称
print(Config.to_dict())       # 获取完整配置字典
```

### 2. NumberParser - 数值解析
```python
from core import NumberParser

# 解析带单位的值
value, unit, original = NumberParser.parse_value_with_unit("10.5 kHz")
# (10.5, "kHz", "10.5 kHz")

# 解析Unicode科学计数法
value = NumberParser.parse_unicode_sci_number("6.6×10⁻⁹")
# 6.6e-09
```

### 3. RangeVerifier - 范围验证
```python
from core import RangeVerifier

# 验证范围
result = RangeVerifier.verify_range_logic("10.5 V", "0~20 V")
# '{"status": "PASS", "reason": "...", ...}'
```

### 4. UnitConverter - 单位转换
```python
from core import UnitConverter

# 判断单位类型
UnitConverter.is_power_unit("dBm")      # True
UnitConverter.is_voltage_unit("mV")     # True
```

### 5. ErrorVerifier - 误差验证
```python
from core import ErrorVerifier

# 验证误差

```

### 6. UncertaintyVerifier - 不确定度验证
